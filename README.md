# Plataforma de E-commerce Custom - Django

Este proyecto es una plataforma de comercio electrónico full-stack desarrollada desde cero con Python y Django. Está diseñada bajo una arquitectura limpia y escalable (MVT), pensada para ofrecer una experiencia de usuario rápida y fluida, ideal para marcas con estética premium o *quiet luxury*.

## Características del Proyecto
* **Catálogo Dinámico:** Filtrado de productos por categorías directamente desde la URL.
* **Sistema de Carrito:** Lógica en backend para agregar productos y calcular totales en tiempo real.
* **Control de Stock:** Barra de progreso matemática que calcula cuánto falta para llegar a metas de envío gratis o stock mínimo.
* **Código Limpio:** Separación estricta de lógica de negocio en `views.py` y maquetación limpia en templates HTML independientes.

## Tecnologías Utilizadas
* **Backend:** Python 3.x, Django Framework
* **Frontend:** HTML5, CSS3, JavaScript (Nativo)
* **Base de datos:** SQLite (Entorno de desarrollo)
